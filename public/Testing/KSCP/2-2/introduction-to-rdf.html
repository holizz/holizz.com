An Introduction to RDF

Abstract

Introduction

In RDF: Concepts and Abstract Syntax , the W3C defines the Resource Description Framework as a framework for representing information in the Web.

The Resource Description Framework (RDF) is a framework for representing information in the Web.

It’s basically a way to make statements about things.

Triples

Each statement consists of a subject, a predicate and an object. For example, in the statement foo eats bar, foo is the subject, eats is the predicate and bar is the object.

Everything’s a URI … except literals

In RDF, all subjects, all objects, and most predicates are Uniform Resource Identifiers (URIs).
