漢字表ツール

このツールで漢字を勉強出来る。ここで表を選んで漢字の音訓や単語を見える。漢字がCSSで隠しているのでセレクトして字が見えてくる。

辞典ファイルはジム・ブリーンのKANJIDICとEDICTです。

ソースをダウンロード出来ます：pyk.tar.bz2

Kanji Display Tool

With this tool you can study kanji. Select a table from here and you can see the readings and words of that character. The kanji are hidden with CSS so you can select the character to see it.

The dictionary files are Jim Breen's KANJIDIC and EDICT.

The source can be downloaded here: pyk.tar.bz2
