Degree

I am currently studying for a BSc in Computer Science at the University of Bradford. These are the first-year modules I am currently studying:

  • Computer Architecture and System Software: Boolean algebra, logic circuits. Interesting, challenging but kind of boring.
  • Formal Foundations: Teaching mathematics related to computing such as discrete mathematics, relations, probability theory and, currently, computer theory (regular expressions and finite automata thus far). Very interesting and rather challenging; not the most exciting lectures but certainly the most interesting.
  • Foundations of Human–Computer Interaction: I studied this last semester. I passed the exam. I got the lowest pass possible (i.e. the lower boundary was 40% and I got 40%).
  • Introduction to Web Technologies: (It’s about the Internet too). This is a terrible module (my colleagues and some staff agree and changes will be made, hopefully). The majority of the course relates to the Web which is a subject I am very well-versed in; it is easy. I get to argue with the lecturer about him forcing us to use tables for layout. The moron. (I had the idea this CV would be slightly professional-looking, but it needed to be said).
  • Key Skills For Computing Professionals: An assortment of things happen in this module. It is not overly-difficult and not overly-useful. (They taught us about brainstorms FFS, this is university not middle school even though half the class may not act it).
  • Software Development (Programming): Teaching object-orientated programming (as if it were the only way). Using the magic bullet known as Java. The only difficult thing is getting Java to work. If they let us hand assignments in in any language this would be piss-easy.
  • Software Development Tools and Design: Relating to various tools to help with programming (including teaching of Emacs, Makefile syntax and UML). Not difficult, but interesting.
