Not-Oddworld: Abe’s Non–Copyright-Infringing Free (Software) Clone

Catchy title, eh?

I am currently pondering a free clone of Oddworld: Abe’s Oddysee/Exoddus. It should certainly be done. I have fond memories of that game. Anybody interested (in writing code, doing artwork or is simply interested in the idea) should contact a user named holizz via a host named gmail.com.

I’m thinking either Python/Pygame for speed, clarity and portability of code or C++/SDL for speed of execution. Possibly even C++/SDL (for the engine) and Python (for the logic). The only problem is that I’ve forgotten most of the C++ I knew and I’ve never used SDL and I’m not that good with Pygame. But I’ll learn what I need to when the language is decided upon.

It should be fun to write, whatever language/library I choose. A rather multidisciplinary endeavour. Games aren’t a field I’ve done much work in previously so it will be exciting.
