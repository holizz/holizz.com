Ratpoison

Right now I am using Ratpoison. It’s a window manager. There are no pretty decorations, no wavy-cool menus popping up everywhere. But it’s special, and do you know why?

Because…

It is largely modelled after GNU Screen

I read that sentence and I knew I wanted it.

Screen is the best thing since, well… computers. I don’t know how I survived before I discovered screen. My first two commands upon logging into X are xterm -e screen and xterm -e screen -x.

I’m writing this entry in vim in screen in an xterm in ratpoison in X11. Sometimes it gets a little confusing navigating all my different things. Spending a long time in screen (like I do every day) sometimes makes me use C-A in Firefox to switch tabs, which doesn’t work (there should be a screen-alike extension for Firefox). It can get tedious when I’m sshed to another host using screen there inside my screen here, which have the same control character because I’m too lazy to change it there.

Don’t forget to check out the pictures of supported hardware (i.e. pictures of keyboards people use with ratpoison). I especially like the datahand and the modified Dvorak Happy Hacking keyboard.
