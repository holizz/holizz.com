I never knew HTTP could be so boring

Okay, so maybe Introduction to Web Technologies today wasn’t that bad. It was mostly covering the basics of HTTP and mentioning some other related protocols.

But Formal Foundations (currently Computer Theory), I rather enjoyed that. Attila Csenki (the lecturer) was talking about regular expressions. But this wasn’t the basics like (?:xyz){6,9}(?=>012), it was an introduction to the theory behind it.

In regular expressions one could say that a* means that it matches 0 or more as in a row. But no, Csenki told us that the closure of a was the set

{Λ, a, aa, aaa, aaaa, aaaaa, …}

(Λ referring to a string of zero length, or "" in most languages).

He seems to be an intelligent person, but I’m not sure he’s too good at teaching first-year idiots who get bored easily and start talking. I like him. He has a pocket watch. All the coolest people have pocket watches. So, uh… That’s Csenki and Owen.
