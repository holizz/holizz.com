Notepad is a minimal GUI text editor. How the hell can it be compared to Emacs, possibly one of the most powerful text editors on the planet (if not the Universe)? I’m not a zealot, mind you. I haven’t used Emacs for a long time. I use Vim, which may not be as powerful but is the most popular editor according to that press release.

I have nothing against people promoting editing with a text editor (I had a friend who couldn’t understand why I used Notepad rather than Word for plain text stuff). But come on, how can I, a Vim user, support a cause with the namesake of a Microsoft product? It’s like having an operating system Web ring called “Windows Webring”. How about the “Text Editor Webring” for a slightly less product-dependant name?

While searching for a button proclaiming something to the effect of “created with XML parser”, I came across an amusing post on a random board: Generated menus not XHTML compliant!.
