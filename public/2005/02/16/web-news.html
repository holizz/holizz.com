Web news

First of all we have a nice article entitled Invisible Information, which discusses the problem with using semantic information that many user agents often ignore. The cite attribute of the blockquote element is the one I’d like to see used more in user agents. I generally manually create a div in the blockquote saying something like “URI: http://…”.
